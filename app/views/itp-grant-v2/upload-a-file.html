Add the weeks of ITP each trainee did

Open the file in spreadsheet software (for example, Microsoft Excel).

In the column ‘Weeks of ITP’, add the number of weeks of ITP each trainee completed.

If a trainee did not complete any ITP enter ‘0’.

If you delete a row the trainee will not be counted towards your claim.

Save or export the file as a ‘csv’.
